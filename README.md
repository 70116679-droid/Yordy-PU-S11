# Pruebas Unitarias con Excepciones en PHPUnit

## Descripción

Este proyecto demuestra el uso de pruebas unitarias en PHP utilizando PHPUnit, enfocándose en la validación de excepciones mediante los métodos:

* expectException()
* expectExceptionMessage()

## Objetivos

* Comprender el manejo de excepciones en PHP.
* Implementar pruebas unitarias con PHPUnit.
* Validar mensajes y tipos de excepciones.
* Aplicar buenas prácticas en el desarrollo de software.

## Tecnologías Utilizadas

* PHP
* PHPUnit
* Git
* GitHub
* Visual Studio Code / NetBeans
